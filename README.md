# Grab_N_Go 
## About The Project
grab N go is contact free store system mocking AmazonGo. 
YOLOv5 + deepSORT + OpenPose + Oracle DB

## Environment
* OS: Ubuntu 20.04
* DB: Oracle
* Tool (Editor Tool): VSCode
* Language: Python 3.8.5

* CUDA version 
* cudnn version  Nvidia 
* Intel Realsense version
